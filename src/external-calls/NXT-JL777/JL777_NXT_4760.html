SuperNET Releases / calculate and send dividends on mainnet
« on: June 01, 2014, 02:29:40 am »
http://209.126.71.170/mainnet_send_dividends.zip

I made a special release for mainnet dividend calculation and automatic payments
It has all the other NXTservices enabled, but only dividend calculations are supported

You can allocate a set amount of NXT or any NXT asset to all asset holders. The issuer is not part of the dividend allocation, with the assumption being that the issuer could have some issued but unbound assets.

You can calculate the asset holders as of any timestamp.

In the directory with the mainnet_send_dividend program, it will create a dividends.csv file. You use this without sending any dividends by not setting the payout field. If you set the payout field to 1, it will create the raw transaction bytes and as long as it is not generating errors and the totals are right, you can use payout value 2

BE CAREFUL. payout value 2 will actually send out the assets or NXT, so you need to make sure the totals are correct before doing this. You need to make sure the NXTservices acct has the required NXT and assets.

The command line window will show a lot of info, including the number of assetholders and total dividend amount. Depending on how many decimals your asset has, it is possible to end up sending more dividend than the specified total. If you want to make sure the total dividend is what you specified, you might have to lower the specified dividend to balance the extra from rounding
