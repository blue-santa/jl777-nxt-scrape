SuperNET Releases / NXTservices_v04 - calc_dividends and acct asset holdings API (testnet +Mac only)
« on: May 25, 2014, 03:33:20 am »
http://209.126.71.170/NXTservices_v04.zip

this is for Mac only

This build has many things active all at once. There might be interaction issues
The new features are the dividend calculation and acct asset holdings API that have been added to NXTcoinsco form that will appear on 127.0.0.1:7777

you can set maxtimestamp in the past and it will ignore any asset transactions after that timestamp
I couldnt find an easy way to map block number to timestamp, so I just used timestamp as it is not germane to the issue

There are two key APIs
Get account history
assetid:    <set to assetid or blank to get all assets>
maxtimestamp 0 for latest:    <blank or 0 for latest, otherwise any tx after maxtimestamp will be ignored>
enable tx log:    <set to 1 if you want to list all transactions, blank if just summary>


Generate dividend list
assetid:    <you have to be nice and put a valid asset id here>
maxtimestamp 0 for latest:    <blank or 0 for latest, otherwise any tx after maxtimestamp will be ignored>


So, the dividend list is mostly the same as last time, but this version, I dont count the assets that are held by the issuer. Also, empty accts are not listed. This should be tested very much as people will be distributing dividends using the JSON from this API.

I also want NXTservices GUI to call the calc_dividend API and then display a the list of NXT accts with a way to delete them from the list. A lot of times there are assets in secondary holding accts, or for whatever reason the dividend is not wanting to be sent.
There should also be a way to specify a Total amount for the still active accounts. Then they say prepare payments and it displays how much NXT would be sent to each acct. If they confirm, then we send the NXT out. Eventually, I need to be able to specify assets to be transferred instead of NXT.

The payments that were not sent out should be saved, preferably in a CSV file so it is easy for asset holders to update NXT acct credit balances when the amount was too small to send.

James
