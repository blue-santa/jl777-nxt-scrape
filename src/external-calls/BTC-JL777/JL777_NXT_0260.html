269  Alternate cryptocurrencies / Announcements (Altcoins) / Re: [ANN][KMD][dPoW] Komodo ICO - Zcash Zero Knowledge Privacy Secured by Bitcoin on: February 05, 2017, 12:09:55 PM
Basilisk seed node issue confirmed, router is being restarted. all should be good in basiliskland in 10 minutes.

this is  a quirk in the basilisk bootstrap, which starts with a seed node and each request gets a random node. It then accepts some (not all) of the returned nodes and increases its list of available notaries to connect to. Over time, eventually all notaries will be connected to.

Once you get even just an additional notary, then the seed node is not needed. Recent gui added an init phase where it established contact.

However, currently for a node that is starting up, it does require the seed node to be active.

What this means is that you are better off not restarting iguana if you are already connected. Also if you get an error when starting up, it is likely just a short term problem and just try again later.

making this seed process a lot more redundant is one of the things on the todo list.
